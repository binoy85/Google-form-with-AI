About the Project

We all used Google Forms many times before. Nowadays, everything is online, from college tests to shopping. Evaluating students through quizzes on Google Forms is effective as the marks are stored there. However, creating quizzes is time-consuming. Many AI tools exist to create quizzes but aren't as efficient or convenient as popular tools like ChatGPT or Google Gemini. This inspired us to create this application.

Development Stages

Initially, we built a command-line application using the Google Forms API with Python. We provided details like form title, description, and questions manually, often generating these questions using ChatGPT. In the next stage, we integrated Google Gemini to generate questions programmatically. Eventually, we developed web pages using Flask, making the application more user-friendly.

Stage 1: Prototype 1 - In this initial stage, we created a basic version of our application using only the Google Form API. Users had to manually input or paste questions and answers into the command prompt. While functional, this process was time-consuming as each question had to be added individually.

Stage 2: Prototype 2 - Building upon the first prototype, we introduced Google Gemini, an AI tool that automatically generates questions based on prompts. This integration allowed for the automatic generation of questions, reducing the manual effort required. However, the interface remained the same, with users still interacting through the command prompt.

Stage 3: Design Implementation - At this stage, we focused on enhancing the user experience by designing the application's interface using Figma. This step involved creating visual mockups and wireframes to improve the overall aesthetics and usability of the application.

Stage 4: Web Interface Development - The final stage involved transforming our command-line application into a user-friendly web page. We revamped the interface to provide a more intuitive and visually appealing experience for users. While implementing the web interface, we made adjustments to ensure consistency between the design and the actual application, resulting in a seamless user experience.

Challenges Faced

We encountered various challenges, with the main issue being consistency. AI-generated questions were not consistently accurate. After researching, we discovered that making the responses in JSON format improved consistency. This resource helped us significantly:Resource Link . However, the response isn't 100% consistent, particularly with the number of generated questions.

Project Workflow

Our project workflow involves the following steps:

  • Input the topics and parameters for the quiz.
  • Generate questions using AI tools.
  • Display the generated questions for user selection.
  • Create a Google Form with the selected questions.
  • Provide a link to the generated Google Form.
